# nicest
nicely integrating complex education software together

## Installation
1. Install [iojs](https://iojs.org/)
2. Install [MongoDb](https://www.mongodb.org/)
3. Run `npm install`

## Start app
1. Run `npm start`

## API Documentation
1. Start app
2. Follow `Documentation` link printed on app startup

## Testing
1. Run `npm test`
